Explanation Pig Game Techniques

DOM

DOM: Document Object Model
Structured representation of an HTML object
The DOM is used to connect webpages to scripts like JavaScript
For each HTML box, there is an object in the DOM that we can access and interact with.

Events

Notifications that are sent to notify the code that something happened on the webpage.
Examples: Clicking a button, resizing a window, scrolling down or pressing a key.
Event Listener: a function that performs an action based on a certain event. It waits for a specific event to happen.

Message Queue

The place where all events that happen in the Browser are put.
They sit here and wait to be processed. This only happens once the Execution Stack is empty.
